# Lista de tarefas

Site de lista de tarefas onde as tarefas adicionadoas ficaram salvas em seu navegador, mesmo atualizando a página.

Resoluções aplicadas ao porjeto:

- 360x800 (mobile);
- 412x914 (mobile);
- XGA (1024x768);
- HD (1280×720);
- WXGA (1366×768);
- HD+ (1600×900);
- FHD (1920×1080).
